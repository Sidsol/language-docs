# C# Language Reference

The language reference provides an informal reference to C# syntax and idioms for beginners and experienced C# and .NET developers.

## Sections

### Reference

- [C# keywords](keywords.md) - Reserved identifiers with special meanings to the compiler
- [C# operators and expressions](operators.md) - Operators for arithmetic, logical, bitwise, equality, and more
- [Built-in types](built-in-types.md) - Value types and reference types built into the language
- [Integral numeric types](integral-numeric-types.md) - Integer types: `int`, `long`, `short`, `byte`, etc.
- [Floating-point numeric types](floating-point-numeric-types.md) - `float`, `double`, and `decimal` types

- [Arithmetic operators](operators-arithmetic.md) - `+`, `-`, `*`, `/`, `%`, `++`, `--`

- [Bitwise and shift operators](operators-bitwise-and-shift.md) - `~`, `<<`, `>>`, `>>>`, `&`, `|`, `^`
- [Equality operators](operators-equality.md) - `==`, `!=`
- [Comparison operators](operators-comparison.md) - `<`, `>`, `<=`, `>=`
- [Lambda expressions](operators-lambda-expressions.md) - Anonymous functions with `=>`
- [Pattern matching](operators-patterns.md) - `is`, `switch`, `and`, `or`, `not` patterns
- [Type testing and cast](operators-type-testing-and-cast.md) - `is`, `as`, `typeof`, cast expressions

### Statements

- [Selection statements](statements-selection.md) - `if`, `if-else`, `switch`
- [Iteration statements](statements-iteration.md) - `for`, `foreach`, `do`, `while`
