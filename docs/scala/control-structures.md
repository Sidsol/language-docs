# Control Structures

Scala has the basic control structures you’d expect to find in a programming language, including:

  - if/then/else
  - for loops
  - try/catch/finally

It also has a few unique constructs, including:

  - match expressions
  - for expressions

We’ll demonstrate these in the following lessons.
